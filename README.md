# INF1002-Python

## Description

Our project is a comprehensive stock analysis web application that uses pre-loaded financial data from multiple sectors (technology, finance, healthcare, consumer goods, and energy) covering the past 2–3 years. The application will allow users to browse stocks by company or sector, view interactive price charts with technical indicators (such as 20-day, 50-day, and 200-day simple moving averages), and analyse performance metrics including daily returns, trend runs, and maximum profit calculations. Additional features include a market overview dashboard (showing top and worst performers, volatility rankings, and trend analysis), comparison tools for up to four stocks, sector performance comparisons over time, and a “Market Moments” feature highlighting stock reactions to major events. All data is pre-processed and embedded, ensuring instant analysis without requiring external uploads.
